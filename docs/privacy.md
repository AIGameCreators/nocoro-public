# Privacy / プライバシー

Nocoro は、PCの中で一緒に過ごす小さな住人AIです。プレビュー版では、監視感の強い動きを避け、必要なときだけそばに出てくる設計を優先しています。

## 何を見るのか

- ユーザーが明示的に「今の画面を見てもらう」を選んだときだけ画面を見ます
- そのほかの時間に、画面を常時監視することはありません
- 今日の観察メモは、作業の流れやPC状態から軽い印象として作ります

## 何を保存するのか

- 今日の観察メモ
- 住人メモ
- 設定内容
- 必要に応じた最小限の状態情報
- 最後にふれた日時や反応頻度など、体験を整えるための軽い状態

## 何を保存しないのか

- 画面キャプチャの恒久保存
- クリップボード本文
- URL本文
- 詳細なテキスト内容
- 個人情報の収集を前提とした記録
- 既存ファイル内容の保存
- キー入力の記録
- ブラウザ履歴の収集

## 外部送信

- ローカル LLM を使う設定では、指定したローカル環境にリクエストします
- 「今の画面を見てもらう」など、画像やAI処理が必要な機能では、実行前にユーザー操作を挟みます
- 天気、ニュース、URL本文取得、クリップボード本文の自動利用は、プレビュー版の標準導線には入れていません
- ローカルで完結する処理は、できるだけローカルで処理します

## 削除

- 保存した内容は後から確認できます
- 不要になった記録は削除できます
- 削除した内容は、アプリ上の表示や今後の反応に使われなくなります

## プレビュー版について

- 一部の仕様や保存形式は今後変わる可能性があります
- 公開初版では、まず安心して使えることを優先します
- 変更がある場合は、README やリリースノートで説明します
