# Install / インストール

## 対応環境

- Windows
- プレビュー版では Windows での動作確認を優先しています

## インストール手順

1. GitHub Releases からインストーラーをダウンロードします
2. ダウンロードしたファイルを実行します
3. 初回起動の案内に従って設定します
4. 必要に応じて反応モードや保存内容の設定を確認します

## 起動時の警告について

- 環境によっては Windows の警告が表示されることがあります
- プレビュー版のため、表示される案内をよく確認してください
- ダウンロード元がこのリポジトリの Releases であることを確認してください

## 初回設定

- 呼んでほしい名前を設定できます
- 反応の強さや頻度を調整できます
- 保存内容やプライバシーに関わる設定を確認できます

## アンインストール

- 通常の Windows アプリと同様に削除できます
- アプリ内で保存した内容がある場合は、必要に応じて先に確認してください

## 困ったとき

- 起動しない場合は、OS、アプリ版、表示されたメッセージを添えて GitHub Issues に報告してください
- 表示位置がずれる場合は、DPI 設定、モニタ構成、スクリーンショットがあると調査しやすくなります
