# FAQ / よくある質問

## これは監視アプリですか

いいえ。常時監視ではありません。画面を見る機能は、ユーザーが明示的に操作したときだけ使う設計です。

## 何が保存されますか

今日の観察メモ、住人メモ、設定内容、反応頻度などの最小限の状態です。

## 観察メモに画面内容は入りますか

詳細な画面内容、クリップボード本文、URL本文、既存ファイル内容は使わない方針です。作業の流れやPC状態から、住人目線の軽いメモとして残します。

## 保存した内容は消せますか

はい。保存内容は確認して削除できます。

## 画面を勝手に見ますか

見ません。ユーザーが明示的に操作したときだけです。

## Windows 以外でも使えますか

今回のプレビュー版は Windows 向けです。

## ローカル LLM が必要ですか

必須ではありません。未接続でも固定文による最低限の反応はできます。AI生成を使う場合は、設定されたローカル環境を使います。

## 勝手にファイルを編集しますか

いいえ。プレビュー版では、ユーザーの既存ファイルを勝手に編集する機能はありません。

## 不具合報告はどこに送ればいいですか

GitHub Issues または X で受け付けます。
