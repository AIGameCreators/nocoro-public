# Known Issues / 既知の制限

プレビュー版で確認している、または起こりうる制限です。

- Windows 向けです。macOS / Linux は今回の公開対象ではありません
- 起動時に警告が出る場合があります
- DPI 設定によって表示位置がずれることがあります
- マルチモニタ環境では、表示位置の見え方が変わることがあります
- スリープ復帰後に表示位置や反応タイミングが不安定になる場合があります
- ローカル LLM が未接続のときは、固定文による最低限の反応になります
- 「今の画面を見てもらう」は手動操作時のみです。常時の画面解析は行いません
- 一部の機能や文言は今後変わる可能性があります

## 報告してほしい情報

- Windows のバージョン
- Nocoro のバージョン
- DPI倍率とモニタ構成
- 何をしたときに起きたか
- 可能であればスクリーンショット
