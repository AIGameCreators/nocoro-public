# Changelog

## [0.1.0] - Preview

Initial preview release of Nocoro / ノコロ.

### Added

- Desktop resident character UI
- Short reactions to time and context
- Onboarding flow
- Reaction mode settings
- Today observation notes
- Resident memos
- Manual screen view flow for the moments when the user asks for it

### Notes

- This is a preview release, so behavior, visuals, and saved data formats may change.
- Windows support is the primary target for this release.
